Information visualization course project

I Build a webpage to help readers learn the celestial movement based on universal gravitation. By using HTML&CSS, I deploy and beautify the webpage to make it clearer and more attractive. Besides, there are many interaction modules provided for users to simulate the circular motion of celestial bodies using JavaScript.

tsunami data downloading and processing platform

The platform is designed to automatically download and process tsunami data. The data is obtained from earthquake and tsunami data website using Website Crawler. After that, I perform some data processing and then built a model that can automatically calculate the required information of tsunami using math algorithm. Finally, by using Python matplotlib and JavaScript, there is map displayed for user.
